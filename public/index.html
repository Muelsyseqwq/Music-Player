关于 yt-dlp

下载 B站 / YouTube 音乐需要安装 yt-dlp 命令行工具， 安装后无需配置即可使用。

运行方式

使用说明
  • B站 / YouTube：需在本机安装 yt-dlp，音频将自动提取为 MP3，并自动匹配歌词
  • 直接链接：支持 mp3、flac、wav、m4a 等音频格式的直链下载
  • 🔍 搜索功能：直接输入歌名搜索B站视频，点击结果即可下载
  • 安装 yt-dlp：pip install yt-dlp 或前往 github.com/yt-dlp/yt-dlp 下载
